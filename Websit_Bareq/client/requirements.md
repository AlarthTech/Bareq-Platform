## Packages
framer-motion | Page transitions and scroll-triggered animations
react-hook-form | Form state management and validation
@hookform/resolvers | Zod validation resolver for forms
lucide-react | Beautiful icons

## Notes
The application must be rendered in RTL (Right-to-Left) direction for Arabic support.
Primary color is set to #F08080 (Light Coral) via CSS variables.
Google Font 'Tajawal' is used for modern, clean Arabic typography.
